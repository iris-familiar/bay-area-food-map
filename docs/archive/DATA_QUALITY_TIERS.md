# 数据质量分级系统

## 分级标准

### 🟢 Tier 1: 完全可信
- 有原始post文件
- 数字可从原始数据计算验证
- 时间线数据完整
- 人工抽查确认

### 🟡 Tier 2: 部分可信
- 有原始post文件
- 部分数据可验证
- 可能缺少时间线

### 🟠 Tier 3: 遗留数据
- 来自旧版本数据库
- 无原始post文件
- 数字无法验证
- 但餐厅信息基本正确

### 🔴 Tier 4: 可疑
- 数字有明显错误
- 时间线不匹配
- 需要重新收集
